Project Overview

Built from scratch using Facebook's Create-React-App, LeagueSearch.GG features fully dynamic rendering of player information returned from the Riot API. Much of this project involves digesting raw data and serving them in a more intuitive and readable form. An example is when a player achieves a kill-death (KD) ratio of 3 or greater, the stats would be colored differently than if the player's match had a KD ratio of 1 or less than 1.

This project used React.js because of their in-built functionality of virtual DOM, making the rendering faster and enhancing the overall user experience. In addition, it makes the application lightweight, leading to boost performance and productivity.
