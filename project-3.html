Jungle Rails

A mini e-commerce application built with Rails 4.2. Features Stripe for payment processing and a administrative page for changing product information.

Live Link

Project Overview

Written in Ruby, Jungle is a mini e-commerce platform built with a administrative dashboard in mind. This allows the administrator of the page to change data on the fly for stock, sales dates and details, and more. Users are also able to create a account to store their purchases.

This website has integration tests written in Capybara and PhantomJS. This creates a seamless experience where disruptions are caught right away and developers can start recitifying bugs immediately.
