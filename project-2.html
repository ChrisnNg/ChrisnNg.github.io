Cat_Park

A interactive heat-map of paid parking spots available in Vancouver cross-mapped against Vancouver Police department's database of documented crimes. Built with Golang

Project Overview

Project Overview

Written as a final project, Cat_Park was built from scratch using Facebook's Create-React-App and features Vancouver Police Department's extensive record of crimes database. It takes data from 534,506 points of documented crimes to plot a heatmap. Adding distinction to the types of crimes, the map can be filtered from general crime to car specific crimes. This is also built in conjunction with Vancouver's Parking Meters API and integrates them altogether to create a map to help with finding safe parking in Vancouver.

In tackling the final project, we saw two pathways to impressing potential employers and making our application stand out. One used the languages we learned within the course and furthering the depth of knowledge we had within it. The other tackled learning a new language we haven't touched and immediately implementing it within our project to show our capabilities to learn. We went with both; enhancing our knowledge in React.js and learning Golang from scratch to implement it.
